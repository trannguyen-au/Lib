WNSlider JQuery Plugin v1.3

This is a demo of WNSlider Jquery plugin. the options available are: 
$("#slider").wnSlider( {  
	timerInterval:6000, 
	pagingType:"image", // "number" display 1-2-3 ...// "none" , no paging display
	autoRotate:true ,  
	effect:"fade", // "none"
	pagingNavigation: true, // display prev-next button
	animationSpeed: 500,  // speed for fade animation
	style: "", //custom style for the slider
});
Html structure: 
	
<div style="float:left; margin-left:10px; width:400px">
	<div id="slider">
		<img src="1.jpg" title="1.jpg" alt="This is testing image" rel="http://www.yahoo.com" style="display:none">
		<img src="2.jpg" title="2.png" alt="this is another testing image" rel="http://www.linkedin.com" style="display:none">
		<img src="3.jpg" title="Third Image" alt="Lorem ipsum dolor amet" rel="http://www.google.com" style="display:none">
	</div>
</div>
