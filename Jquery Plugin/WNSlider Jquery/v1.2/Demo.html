WNSlider JQuery Plugin v1.2

This is a demo of WNSlider Jquery plugin. the options available are: 
$("#slider").wnSlider( {  
	timerInterval:6000, 
	pagingType:"image", // "number" display 1-2-3 ...// "none" , no paging display
	autoRotate:true ,  
	effect:"fade", // "none"
	pagingNavigation: true, // display prev-next button
	animationSpeed: 500,  // speed for fade animation
	style: "", //custom style for the slider
});
