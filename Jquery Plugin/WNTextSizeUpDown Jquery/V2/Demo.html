WNTextSizeUpDown Jquery Plugin v1.0

Larger | Smaller

Can you change this p tag size?

This is a span tag
This is a div, class = "text-zize"
This is a content inside a table with id = test_table

Usage:

$(document).ready(function() {
	$("#textSizeChange").wnTextSizeUpDown({option:value});
});

[option]		: [value]
---------------------------------------------
selector 		: ["body"],  // an array of jquery selector on which text-size will be changed, eg: ["div", "#largerText", "input[type='text']"] ....
largerText		: "Larger Size",  // text to be displayed for larger text trigger
smallerText		: "Smaller Size",  // text to be displayed for smaller text trigger
seperator		: "|",			// seperator between the 2 link buttons
times			: 5		// number of times that the font-size can be increased
